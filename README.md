# Homebrew Tap for Capivv

Homebrew formulae for [Capivv](https://capivv.com) — Revenue as Code for in-app subscriptions.

## Install

```bash
brew tap capivv/tap
brew install capivv
```

## Upgrade

```bash
brew upgrade capivv
```

## Uninstall

```bash
brew uninstall capivv
brew untap capivv/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `capivv` | CLI for managing in-app subscriptions with Capivv |

## Other Installation Methods

| Method | Command |
|--------|---------|
| npm | `npx @capivv/cli` |
| Direct download | [GitHub Releases](https://github.com/capivv/homebrew-tap/releases) |
